# SMART RESTAURANT MANAGEMENT SYSTEM

A web-based Restaurant Management System developed using HTML, CSS, JavaScript, Python Flask, and SQLite.

## Features
- Restaurant Menu Management
- Online Food Ordering
- Customer Order Processing
- SQLite Database Integration
- Responsive User Interface

## Technologies Used
- HTML
- CSS
- JavaScript
- Python Flask
- SQLite

## Project Structure
SMART-RESTAURANT-MANAGEMENT-SYSTEM/
├── Flask/
├── database/
├── HTML/
├── CSS/
├── README.md
└── requirements.txt
